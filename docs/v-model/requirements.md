# Requirements Specification

## 작성 규칙

- 각 요구사항은 고유 ID를 가진다.
- 기능과 구현 방법을 분리하고 관찰 가능한 판정 기준을 포함한다.
- 변경 시 [traceability-matrix.md](traceability-matrix.md)의 설계, 코드, 시험 연결을 함께 검토한다.

## 이해관계자 요구사항

| ID | 요구사항 | 확인 기준 |
| --- | --- | --- |
| WC-STK-001 | 사용자는 조이스틱으로 휠체어의 전진, 후진과 좌우 방향을 명령할 수 있어야 한다. | 네 방향 명령과 중립 정지가 구분된다. |
| WC-STK-002 | 진행 방향에 장애물이 존재하면 구동 명령보다 안전 정지가 우선되어야 한다. | 전·후방 장애물 입력에 대응하는 방향의 모터가 정지한다. |
| WC-STK-003 | 사용자는 구동 제어를 명시적으로 활성화하거나 비활성화할 수 있어야 한다. | 버튼 상태 전환과 비활성 상태의 모터 정지를 확인한다. |
| WC-STK-004 | 경사와 진동이 있는 환경에서도 자세 데이터가 안정적으로 처리되어야 한다. | 필터 적용 전후 자세 데이터의 변동을 비교한다. |

## 시스템 요구사항

| ID | 시스템 요구사항 | 할당 | 판정 기준 |
| --- | --- | --- | --- |
| WC-SYS-001 | 시스템은 조이스틱 X/Y축을 독립된 ADC 채널로 취득해야 한다. | STM32 | ADC1 IN10/IN11 값이 갱신된다. |
| WC-SYS-002 | 시스템은 조이스틱 중립 영역에서 모터 방향 출력을 정지 상태로 설정해야 한다. | STM32 | X=3000~3200, Y=2900~3200 경계 조합을 확인한다. |
| WC-SYS-003 | 전방 장애물 입력이 활성화된 상태에서는 전진 명령을 차단해야 한다. | Raspberry Pi/STM32 | PB10 활성 및 전진 명령에서 네 방향 GPIO가 정지 상태가 된다. |
| WC-SYS-004 | 후방 장애물 입력이 활성화된 상태에서는 후진 명령을 차단해야 한다. | Raspberry Pi/STM32 | PB4 활성 및 후진 명령에서 네 방향 GPIO가 정지 상태가 된다. |
| WC-SYS-005 | 조이스틱 제어가 비활성화되면 다음 제어 반복에서 모터 방향 출력을 초기화해야 한다. | STM32 | PA0 토글 후 PA5/PA6/PA9/PC7이 LOW가 된다. |
| WC-SYS-006 | 좌우 모터는 독립된 PWM 채널과 방향 GPIO로 제어되어야 한다. | STM32/Motor Driver | TIM2 CH2와 TIM3 CH2 및 방향 핀 매핑을 확인한다. |
| WC-SYS-007 | 자세 측정값에는 이동평균필터를 적용하고 필터링된 값으로 수평 보정 명령을 생성해야 한다. | Raspberry Pi/STM32 | 필터 출력과 리니어 액추에이터 명령의 연결을 확인한다. |
| WC-SYS-008 | 센서값과 제어 상태를 진단 인터페이스에서 확인할 수 있어야 한다. | STM32 | USART3 115200 baud 출력으로 상태를 식별한다. |

## 소프트웨어 요구사항

| ID | 소프트웨어 요구사항 | 판정 기준 |
| --- | --- | --- |
| WC-SWR-001 | ADC 입력 필터는 최근 8개 샘플의 산술평균을 출력해야 한다. | 8개 이하와 초과 입력 시 기준 계산값과 일치한다. |
| WC-SWR-002 | 필터 초기 구간은 입력된 샘플 수만 분모로 사용해야 한다. | 100, 200, 300 입력의 출력이 100, 150, 200이다. |
| WC-SWR-003 | 장애물 인터록은 조이스틱 방향 판정 후 모터 명령보다 먼저 평가되어야 한다. | 활성 인터록에서 방향 분기 코드가 실행되지 않는다. |
| WC-SWR-004 | 조이스틱 비활성 상태에서는 ADC 기반 방향 명령을 실행하지 않아야 한다. | 비활성 상태에서 모터 정지 후 다음 반복으로 이동한다. |
| WC-SWR-005 | 전·후방 장애물 상태는 서로 배타적으로 갱신되어야 한다. | 전진, 후진, 중립에서 rxDataF/rxDataB 조합을 확인한다. |
